Для регистрации нужно ввести действующий email, так как если забудете пароль, на эту почту придёт ссылка для сброса пароля. После регистрации вы попадёте на страницу логина и нужно ввести email и пароль, не спрашивайте почему - просто так надо. Ссылку на Телеграмм-бота увидите в меню. Для привязки Телеграмм-бота к аккаунту нужно ввести в боте комманду /login. Бот пришлёт ссылку, перейдите по ней и увидите код для привязки - введите этот код в чат Телеграмм-бота.

Меню сервиса:

Новая запись
Сделать новую запись (пациент, клиника, вид работы, количество, дата назначения, комментарий). Если нужно добавить несколько видов работ, сначала сохраните запись с одним пунктом, потом добавьте новую работу.

Список записей
Открыть список записей. По умолчанию откроются все актуальные записи (у которых дата назначения не раньше текущего месяца и статус не "ОПЛАЧЕНО"). Если нужно открыть только записи текущего месяца, откройте нижнее меню, нажмите "другой месяц" и выберете текущий месяц.

Карта работ
Список видов работ (название, цена). Эти работы будут отображаться на странице "НОВАЯ ЗАПИСЬ" в поле "вид работы" и в качестве колонок таблицы на странице "СПИСОК ЗАПИСЕЙ". По указанным ценам будет считаться доход.

Профиль
Ваши учётные данные (имя, email, дата регистрации). Также на этой странице можно поменять свои данные.

Для изменения цены работы, введите новое значение и нажмите "сохранить". Или нажмите "отмена", чтобы закрыть редактирование.

Для изменения данных записи (пациент, клиника, статус, дата назначения), Нажмите на значение, которое нужно поменять, введите новое и нажмите ✅ для сохранения изменений, или ❌ чтобы закрыть редактирование. Чтобы добавить новый вид работы, нажмите "новая работа". Если добавить вид работы, который уже есть в этой записи, то к существующей работе просто добавится новое количество. Дата назначения есть у кажного вида работы, общая дата назначения всей записи устанавливается по самой поздней дате назначения из списка видов работ. Но можно установить общую дату назначения вручную. По общей дате назначения сортируются записи по месяцам. Чтобы добавить/посмотреть фото записи - нажмите кнопку "ОТКРЫТЬ" в строке с количеством фотофайлов.

Список записей. Цвет строк зависит от статуса записи. Для просмотра информации о записи, нажмите на нужную строку. Для дополнительных опций, нажмите нижнее меню.

Посчитать доход
Считает сумму всех видов работ в каждой записи за выбранный месяц, по ценам в карте работ

Поиск записи
Найти все соответствующие ваши записи по введёднным критериям. Можно ввести либо оба поля (пациент и клиника), либо какое-то одно.

Другой месяц
Список записей за указанный месяц. Учитывается значение общей даты назначения записи.

Скачать
Скачать отчёт в excel файле за выбранный месяц

Сортировать:

По текущему месяцу
Сортирует список актуальных работ (дата назначения не раньше текущего месяца и статус не "ОПЛАЧЕНО"). Если дата назначения записи не позже текущего месяца и статус "ДЕЛАЕТСЯ" - установится статус "ВЫПОЛНЕНО".

По прошлому месяцу
Сортирует список актуальных работ (дата назначения не раньше текущего месяца и статус не "ОПЛАЧЕНО"). Если дата назначения записи не позже прошлого месяца и статус "ДЕЛАЕТСЯ" - установится статус "ВЫПОЛНЕНО".

Выбрать несколько
Выбрать несколько записей и установить статус вручную.

Выбрать несколько записей и установить нужный статус. Можно выбрать сразу все, нажав в шапке таблицы (указано зелёной стрелкой на картинке). Цвета строк также отображают статусы записей.

Страница профиля пользователя. Свои данные вы можете изменить, нажав на нужное значение. Для смены email вам придёт на почту ссылка на страницу ввода нового email. Просто следуйте указаниям после нажатия кнопки. Если нет доступа к email, можно сменить с помощью Телеграмм, придёт код для установки нового email. А если не привязан Телеграмм и нет доступа к email - сорян... Не знаю как помочь. Обратитесь к разработчику. Для смены пароля просто введите старый пароль и новый (дважды, чтобы было надёжнее). Если нужно, можно удалить профиль - все данные будут удалены без возможности восстановления. Подумайте хорошо, прежде чем удалять.
